## 一、产品概述

### 1.1 产品定位
一款面向**刚毕业大学生**的像素风格菜谱工具，帮助厨房新手解决「不知道做什么→快速匹配菜谱→跟着教程做→记录饮食成长」的全流程。

### 1.2 目标用户
| 维度 | 描述 |
|------|------|
| 核心用户 | 刚毕业 1-3 年的大学生 |
| 生活状态 | 租房/自住，有独立厨房，烹饪新手 |
| 核心痛点 | 不知道吃什么、想做但不会做、想记录但没工具 |
| 使用场景 | 下班后做晚饭、周末下厨、朋友来聚会 |

### 1.3 技术形态
| 决策 | 说明 |
|------|------|
| 形式 | **浏览器 PWA**（网址打开即用，可保存到手机桌面） |
| 兼容 | iOS Safari / Android Chrome 均可，无需下载安装 |
| 离线 | 核心菜谱浏览和教程可离线使用 |
| 数据 | 饮食记录等数据存储在手机本地，不上传服务器 |
| 交付 | 完整代码开发 + 帮助部署上线，用户拿到网址直接用 |
| 菜谱图片 | 像素风小图标（类似星露谷物语食物icon风格），50道菜各一个 |

### 1.4 参考风格
- 整体风格参考：星露谷物语（Stardew Valley）的UI和食物图标
- 界面感觉：像一款做饭经营养成小游戏
- 色彩：暖色像素调色板，给人温暖治愈的感觉

## 二、视觉风格

### 2.1 整体风格：像素风（Pixel Art）
- 复古 8-bit/16-bit 游戏风格
- 像素字体、像素图标、像素边框和装饰
- 食物用像素小图标呈现
- 界面像一款「做饭经营养成游戏」

### 2.2 配色方案：暖色系
| 色值 | 用途 |
|------|------|
| `#FF6B35` 暖橙 | 主色调，按钮/强调 |
| `#FFB347` 暖黄 | 辅助色，高亮/标签 |
| `#FFF5EE` 暖白 | 背景色 |
| `#FFE4CC` 浅暖橙 | 卡片背景 |
| `#5D4037` 深棕 | 主文字 |
| `#8D6E63` 浅棕 | 次要文字 |
| `#FF8A65` 柔和橙 | 进度条/图表 |
| `#66BB6A` 像素绿 | 成功/达标 |
| `#EF5350` 像素红 | 未达标/提醒 |

### 2.3 字体
- 中文：像素风格中文字体（如 zpix / 方舟像素字体）
- 数字/英文：像素等宽字体（如 Press Start 2P / VT323）

### 2.4 设计元素
- 按钮：像素化边框，按下有「像素位移」效果
- 卡片：像素边框 + 轻微像素阴影
- 图标：定制像素小图标（🍳🥘🥬 等食物的像素版）
- 分割线：像素波浪/锯齿线
- 加载状态：像素角色走路动画
- 导航栏：像素化底部标签栏

## 三、功能模块详情

### 模块1：智能菜谱匹配 🔍

| 编号 | 功能点 | 说明 |
|------|--------|------|
| M1.1 | 食材输入 | 文字输入食材（用逗号/顿号分隔）或点击标签选择 |
| M1.2 | 常用食材快捷选择 | 列出最常用的 20 种食材，点一下即可添加 |
| M1.3 | 三级推荐结果 | 🥇经典必做（高度匹配+经典组合）→ 🥈换个口味（高度匹配其他菜）→ 🥉凑合能吃（缺1-2样，并告知缺什么） |
| M1.4 | 懒人模式 | 输入一句话需求，自动匹配：「想吃辣的，简单的，20分钟内」 |
| M1.5 | 缺食材一键采购 | 匹配结果中显示缺少的食材，一键加入采购清单 |
| M1.6 | 匹配历史 | 记录最近的食材输入，可快速复用 |

### 模块2：菜谱库 📖

| 编号 | 功能点 | 说明 |
|------|--------|------|
| M2.1 | 中餐/西餐分栏 | 两个独立Tab，各自浏览 |
| M2.2 | 筛选条件 | 难度（简单/中等/困难）、时间（<15/<30/<60分钟）、菜系、场景 |
| M2.3 | 场景分类 | 下班快手菜、周末仪式感、朋友聚会、一人食 |
| M2.4 | 搜索 | 按菜名、食材名、标签搜索 |
| M2.5 | 排序 | 综合推荐 / 最多人做过 / 好评最多 |
| M2.6 | 菜谱详情页 | 食材清单 + 分步教程 + 营养信息 + 小贴士 + 替代方案 |
| M2.7 | 相似推荐 | 每道菜底部推荐相关菜谱 |

### 模块3：菜谱互动 ⭐

| 编号 | 功能点 | 说明 |
|------|--------|------|
| M3.1 | 收藏菜谱 | ⭐ 收藏到个人收藏夹 |
| M3.2 | 点赞教程 | 👍 对菜谱的教程质量点赞 |
| M3.3 | 标记做过 | ✅ 「我做过这道菜」，计入做饭记录 |
| M3.4 | 写心得笔记 | 📝 做完后写一句话笔记（如「糖减半更好吃」） |
| M3.5 | 记录实际用时 | ⏱️ 记录自己实际花了多久，积累个人数据 |

### 模块4：饮食追踪 + 营养分析 📊（核心功能）

| 编号 | 功能点 | 说明 |
|------|--------|------|
| M4.1 | 记录一顿饭 | 选菜谱 → 选餐别（早/午/晚/加餐）→ 填份量 → 自动算营养 |
| M4.2 | 每日营养仪表盘 | 环形进度条：热量/蛋白质/碳水/脂肪，目标完成度可视化 |
| M4.3 | 营养目标自定义 | 用户可设置自己的每日热量、蛋白质等目标值 |
| M4.4 | 营养缺口提醒 | 某项不达标时智能提醒（如「蛋白质还差15g，晚餐可以加个鸡蛋」） |
| M4.5 | 日/周/月/年视图 | 切换时间维度查看营养数据 |

### 模块5：做饭日记 📝（音乐App周榜月报风格）

| 编号 | 功能点 | 说明 |
|------|--------|------|
| M5.1 | 周报生成 | 每周自动总结：做了几道菜、Top排行榜、营养趋势、省钱计算 |
| M5.2 | 月报生成 | 每月深度总结：菜谱排行榜Top 10、菜系偏好、食材频率、成长曲线 |
| M5.3 | 年报生成 | 年度回顾：累计下厨次数、解锁菜谱数、年度最爱、年度省钱总额 |
| M5.4 | 菜谱排行榜 | 统计做最多的菜（回头率排行） |
| M5.5 | 菜系偏好分析 | 川/粤/鲁/意/法...各做了多少次，口味地图分布 |
| M5.6 | 食材使用频率 | 鸡蛋用了多少次、西红柿用了多少次... |
| M5.7 | 下厨频率趋势 | 本周vs上周、本月vs上月的做饭次数对比 |
| M5.8 | 频次热力图 | 类似GitHub绿格，展示本月每天做饭情况 |
| M5.9 | 省钱计算器 | 自己做饭花费 vs 外卖均价，算出省了多少钱 |
| M5.10 | 数据导出/分享 | 生成精美的像素风月报长图，可保存或分享朋友圈 |

### 模块6：采购清单 🛒

| 编号 | 功能点 | 说明 |
|------|--------|------|
| M6.1 | 自动生成 | 从匹配结果「缺的食材」或菜谱食材一键添加 |
| M6.2 | 分类显示 | 蔬菜/肉蛋/调料/主食 分组 |
| M6.3 | 勾选已购 | 买完划掉 |
| M6.4 | 清空已完成 | 一键清除已勾选项 |

### 模块7：个人中心 👤

| 编号 | 功能点 | 说明 |
|------|--------|------|
| M7.1 | 我的收藏 | 收藏的菜谱列表 |
| M7.2 | 我的心得 | 做菜笔记汇总 |
| M7.3 | 成就系统 | 游戏化徽章（新手/达人/省钱王等） |
| M7.4 | 厨房装备设置 | 勾选自己的厨具，只推荐能做的菜 |
| M7.5 | 口味偏好 | 辣度/忌口/过敏源 |
| M7.6 | 营养目标设置 | 自定义各营养素目标值 |

### 模块8：辅助功能 🎯

| 编号 | 功能点 | 说明 |
|------|--------|------|
| M8.1 | 配方调整器 | 输入N人份，自动换算食材用量 |
| M8.2 | 季节性推荐 | 首页根据季节推荐应季菜谱 |
| M8.3 | 今日推荐 | 打开App随机推荐一道菜，想不好吃什么时用 |
| M8.4 | 食材保鲜小知识 | 菜谱食材旁附保存方法（如「西红柿常温保存不要放冰箱」） |

## 四、数据规划

### 4.1 内置菜谱：50道
| 类别 | 数量 | 包含 |
|------|------|------|
| 中餐 | 25道 | 涵盖蛋类、猪肉、鸡肉、牛肉、豆腐、蔬菜、米饭/面食、汤类 |
| 西餐 | 15道 | 涵盖意面、沙拉、三明治、简单烘焙、一锅炖 |
| 基础汤/甜品 | 10道 | 紫菜蛋花汤、番茄蛋汤、银耳汤、简单甜品等 |

### 4.2 菜谱数据结构（每条菜谱包含）
- 菜名、所属分类、场景标签、emoji/像素图标
- 食材清单（含用量、是否可选、替代方案）
- 分步教程（每步含文字说明、可选计时器提醒）
- 难度（1-3）、制作时间、口味评分
- 每份营养成分（热量/蛋白质/碳水/脂肪/纤维）
- 厨艺小贴士

### 4.3 用户数据（本地存储）
- 饮食记录（每天每顿吃了什么）
- 收藏菜谱列表
- 做菜心得笔记
- 成就进度
- 采购清单
- 个人设置（营养目标、口味偏好、厨具装备）

## 六、功能优先级（分阶段实现）

### 第一阶段：核心必备 🥇
- 菜谱库浏览（中餐/西餐分栏 + 搜索筛选）
- 菜谱详情页（食材+教程+营养信息）
- 智能食材匹配（三级推荐）
- 收藏菜谱
- 基本饮食记录（记录一顿饭 + 每日营养展示）

### 第二阶段：核心升级 🥈
- 做饭日记（周报 + 月报）
- 采购清单
- 点赞教程、标记做过、写心得
- 营养目标自定义 + 缺口提醒
- 成就系统
- 食谱数据完善至50道

### 第三阶段：体验升级 🥉
- 懒人模式
- 数据导出/分享长图
- 配方调整器（N人份换算）
- 季节性推荐
- 食材保鲜小知识
- 像素动画和微交互打磨
